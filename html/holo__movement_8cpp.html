void subscriberCallback ( const nav_msgs::Odometry::ConstPtr &  pose_msg)

Callback publishing the velocity after reading a position

This function computes the velocity published on the topic "/cmd_vel" based on the estimated position retrieved from the messages in the "/odom" odometry topic. When the target position is reached a call to the Service "/target_pos" is made in order to retrieve a new target position.

Parameters
pose_msg(/nav_msgs::Odometry::ConstPtr&): pointer to a message read from topic "/odom", used to obtain estimated current position and, if the flag 'pseudo_inertia' = 'true', estimated current linear velocity;

int vel_algorithm = 0

Flag determining the criteria used for evaluating the velocity

  • 0: velocity is directly proportional to current distance from target position;
  • 1 : velocity is the weighted sum between one component, being proportional to current distance from target position and the previous velocity
